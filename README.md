# LoadStatePager
A library to load fragment page with different state

## Usage
```
Gradle:
implementation 'com.curious.loadstate:loadstate:1.0.1'
```

```
Maven:
<dependency>
  <groupId>com.curious.loadstate</groupId>
  <artifactId>loadstate</artifactId>
  <version>1.0.1</version>
  <type>pom</type>
</dependency>
```
